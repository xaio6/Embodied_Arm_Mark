# 常见问题与故障排查

## 连接与通信

### 无法连接电机（串口打开失败）

检查项：
- 串口号是否正确（Windows 设备管理器）
- 是否被其他程序占用
- USB 转 CAN 设备驱动是否正常

### 电机无响应

检查项：
- 驱动板 CAN 速率是否与程序一致
- 电机 ID 是否正确且不冲突
- 是否已经使能

## 运动控制

### 运动到位判断不稳定

建议：
- 避免多个程序同时读写同一条总线
- 在高频控制场景中，将读状态与写命令做串行化

## 视觉与相机

### 摄像头打不开

检查项：
- `camera_id` 是否正确
- 摄像头是否被其他程序占用
- OpenCV 是否安装完整

### 抓取位置明显偏差

检查项：
- 是否使用了匹配当前相机/安装位姿的标定文件
- 是否正确设置 TCP 偏移与抓取深度参数

## 仿真

### 仿真无法启动

检查项：
- 是否安装 `mujoco`（仅在你需要仿真时才需要）：`pip install mujoco`
- 显卡与 OpenGL 环境是否可用
- 模型文件是否存在：`config/urdf/mjmodel.xml`

参考：
- `example/mujoco_control.py`
- `example/docs/simulation.md`
